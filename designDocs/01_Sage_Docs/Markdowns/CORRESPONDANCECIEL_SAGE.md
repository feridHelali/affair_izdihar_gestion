## <span id="page-8-0"></span>Introduction

Ce chapitre récapitule les informations de Ciel Comptabilité récupérées lors de la conversion des dossiers. Les zones non gérées par Sage Comptabilité sont ignorées. Pour les zones non gérées dans Ciel Comptabilité, la valeur par défaut attribuée lors de la conversion est précisée.

Les fichiers sont convertis dans la structure des bases de la ligne 100.

#### Traitement des exercices N-1

Dans Ciel, lors de la création d'un nouvel exercice : l'exercice précédent est archivé, les détails des lignes d'écritures sont supprimés. Seuls les cumuls de l'exercice N-1 sont conservés à des fins de comparatifs N / N-1 lors de l'impression de la balance.

Lors de la conversion du fichier Ciel, les soldes N-1 sont récupérés en tant que balance de clôture au dernier jour de l'exercice N-1, avec création d'une ligne d'équilibre sur le compte 8900.

## Particularités

## Rapprochement bancaire

Le rapprochement bancaire peut être effectué par la fonction Pointage ou par le Rapprochement bancaire dans Ciel. Selon le cas, le rapprochement est récupéré :

- Pointage (le champ Lignes d'écritures / OrdreRapBq est alors nul) : le code pointage est récupéré dans la zone N° pièce de trésorerie,
- Rapprochement bancaire (le champ Lignes d'écritures / OrdreRapBq est alors renseigné) : le lien avec le champ Rapprochements bancaires / Ordre permet de récupérer la Référence saisie lors du rapprochement.
